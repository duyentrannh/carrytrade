A3¶

We see that the profit margins in question 2A and 1A remained unchanged. There are two effects.

  • Inflation impact: high inflation in the US means that the price of Norwegian company's phone in the US market will increase 5%, hence the revenue(USD)grow at 5% rate. While the cost (NOK) grow only at 1% rate due to Norwegian have lower inflation rate.
  • Exchange rate Impact: The NOK appreciates againts the US dollar, from 8 to 7.6952 NOK/USD. It means hat each USD is worth fewer NOK than in 1A. For a Norwegian company that earns revenue in USD, this appreciation would be unfavorable when convert USD to NOK because you will get fewer NOK per USD.

However, with the inflation impact that the revenue in USD increase due to the inflation in the US and Revenue in NOK is impacted by the appreciation of NOK against USD. These two effects balance each other, so the profit margin maintains constant.

A4¶

The financial deparment came up with such estimate of the exchange rate in year t + 1, using the concept of relative purchasing power parity(RPPP). They believe that RPPP will hold.

$ RPPP_t = S_{t-1} * \frac {1 +inflation_t} {1 + inflation_t^*} $

The results below show that the RPPP is the same as expected exchange rate. Hence ,RPPP shows that countries with higher inflation will experience currency depreciation, thus leaving the real exchange rate constant.

exchange_rate = 8.0  # NOK/USD
inflation_norway = 0.01   # Percentage
inflation_usa = 0.05     # Percentage

def calculate_rppp_exchange_rate(exchange_rate, inflation_norway, inflation_usa):
    """
    Calculate the new exchange rate based on Relative Purchasing Power Parity (RPPP).
    
    Parameters:
    exchange_rate (float): The current exchange rate (Home currency per Foreign currency)
    inflation_norway (float): The inflation rate in the home country (as a percentage)
    inflation_usa (float): The inflation rate in the foreign country (as a percentage)
    
    Returns:
    float: The new exchange rate as predicted by RPPP.
    """   
    # Calculate the new exchange rate using RPPP formula
    rppp= exchange_rate * ((1 + inflation_norway) / (1 + inflation_usa))
    
    return rppp

def create_portfolios(df3, total_portfolios=6):

    # Initialize a dictionary to store portfolio assignments with the date 
    portfolio_assignments = {'Date': df3['Date'].tolist()}
    # Add portfolio keys to the dictionary
    for i in range(total_portfolios):
        portfolio_assignments[f'Portfolio {i+1}'] = []

    for index, currency_row in df3.iterrows(): 
        # Rank the currencies in the row, excluding NaN values and the 'Date' column
        ranked_currencies = currency_row.dropna().drop('Date').rank(method='min', ascending=True)
        # If there are not enough currencies for the portfolios, add empty lists for this date
        if ranked_currencies.empty or len(ranked_currencies) < total_portfolios:
            for i in range(total_portfolios):
                portfolio_assignments[f'Portfolio {i+1}'].append([])
            continue  # Then countinue

        # Determine the rank boundaries for dividing currencies into portfolios
        rank_boundaries = np.percentile(ranked_currencies.values, np.linspace(0, 100, total_portfolios + 1))

       # Assign each currency to a portfolio based on its rank
        for portfolio_num in range(total_portfolios):
            # Create a condition to check if a currency's rank falls within a portfolio's boundary
            # For the first portfolio (Portfolio_1), the condition includes currencies with ranks 
            # up to the first boundary (inclusive)
            if portfolio_num == 0:
                inclusion_condition = ranked_currencies <= rank_boundaries[portfolio_num + 1]
                 # For other portfolios, include currencies within specific rank ranges
            else:
                inclusion_condition = (ranked_currencies > rank_boundaries[portfolio_num]) & \
                                      (ranked_currencies <= rank_boundaries[portfolio_num + 1])
                # Get the list of currencies that fit the criteria for this portfolio
            assigned_currencies = ranked_currencies[inclusion_condition].index.tolist()
            # Add these currencies to the current portfolio
            portfolio_assignments[f'Portfolio {portfolio_num+1}'].append(assigned_currencies)

    # Convert the dictionary to a DataFrame
    portfolio_df = pd.DataFrame(portfolio_assignments)
    
    return portfolio_df
